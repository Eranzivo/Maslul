# הקשר עסקי - Dona Real Estate

> מקור: אתר החברה הרשמי [dona.co.il](https://www.dona.co.il/). נעשה שימוש במידע רלוונטי בלבד, ללא העמסה.
> פרטים תפעוליים מדויקים (שעות מוקד, שמות מחלקות) - **טעון אימות מול דונה לפני שימוש מול לקוח**.

## מי זו דונה

חברת נדל"ן ובנייה ותיקה ומובילה בישראל, פעילה **למעלה מ‑30 שנה**. לפי האתר: כ‑**16,000 יחידות דיור**
שנבנו, **למעלה מ‑100 פרויקטים** ברחבי הארץ, ודירוג גבוה ממשרד הבינוי והשיכון.

**המסר המרכזי:** *"דונה זה בית"* - ולצידו תפיסת השירות *"דונה עם הפנים לדייר"*. השפה באתר חמה, משפחתית,
שמה דגש על איכות חיים, ליווי אישי לאורך כל הדרך, וביטויים כמו *"הגעתם הביתה"*. זו נימת המותג שכל סוכן AI חייב לשמר.

| מגזר | תיאור |
|---|---|
| **מגורים** | פרויקטים בשיווק ופרויקטים שהושלמו |
| **השכרה** | דירות להשכרה לטווח ארוך |
| **מסחרי** | נדל"ן מסחרי (כ‑55,000 מ"ר) |
| **התחדשות עירונית** | פינוי‑בינוי / תמ"א - מתחמים ותיקים |

**דוגמאות לפרויקטים בשיווק** (לשימוש כדוגמאות ל‑Project Memory): דונה TOP רמלה צפון, דונה CROWN מודיעין,
דונה בכרמי גת, דונה בפסגה (ירושלים).

## נקודות שירות שרלוונטיות ישירות לסוכנים

- **מוקד שירות / קריאות שירות:** `*2966` + **מחלקת שירות דיירים** - זהו יעד ההסלמה הטבעי של **סוכן בדק הבית**
  (פתיחת קריאת שירות, מעבר לטיפול אנושי).
- האתר מדגיש **ליווי אישי**, "שירות מצוין ומקצועי בכל שלב", ויחס אישי לדייר - בדיוק חוויית המשתמש שהמשימה דורשת.
- **מסלול שירות / בדק** קיים באתר - מתחבר ישירות לסוכן בדק הבית (תקופת בדק/אחריות, פתיחת קריאה).

## מסע הלקוח (Customer Journey) ואיפה כל סוכן נכנס

```
התעניינות/רכישה → חתימת חוזה → בנייה → מסירת דירה → מגורים ואחריות
     │                  │            │           │              │
  (לא בתחום        סוכן 2:       סוכן 3:      סוכן 2:        סוכן 1:
   3 הסוכנים)      משפטי/חוזים   קבלנים/ביצוע  מסמכי מסירה    בדק בית
                                              סוכן 1+2:      (ליקויים,
                                              ליווי מסירה    אחריות,
                                                            קריאות שירות)
```

- **רוכש דירה** → סוכן משפטי (הבנת חוזה/סעיפים) + לאחר מסירה סוכן בדק בית.
- **דייר אחרי מסירה** → סוכן בדק בית (ליקויים, אחריות, קריאת שירות) - הלב של "דונה עם הפנים לדייר".
- **קבלן / איש ביצוע / קבלן משנה** → סוכן קבלנים (מושגים, נהלים, תיאום עבודות).

## עקרונות תכן רלוונטיים לדונה

- **חוויית שירות בזמן אמת:** ערוץ WhatsApp, תיעדוף לפי דחיפות, מעקב בוט‑מול‑אדם, והסלמה עם הקשר מלא -
  מתאימים מצוין לדייר הישראלי שחי ב‑WhatsApp ולתפיסת השירות של דונה.
- **ארכיטקטורת סוכנים מתוזמרת:** Orchestrator + רכיבים (ניתוב, שפה, זיכרון, הסלמה, כלים, QA, Observability,
  Compliance) - שכבת התזמור שתוכננה כאן (ראו `orchestration/`).

## הערה על שפה

קהל הדיירים/רוכשים בישראל מגוון (עברית, ערבית, רוסית, אנגלית). הפרומפטים כתובים **בעברית** (דרישת המשימה),
אך רכיב השפה (`orchestration/language.md`) מאפשר מענה בשפת המשתמש - יתרון שירותי אמיתי (תמיכה רב‑לשונית).
