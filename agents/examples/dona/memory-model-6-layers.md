## vector מול episodic

- **Episodic (זיכרון אפיזודי):** מהלך השיחה והפריטים הפתוחים - "מה קרה ומתי".
- **Vector (אחזור סמנטי):** שליפת ידע/Vault רלוונטי לפי משמעות (RAG) - מזין את תשובות הסוכן.
